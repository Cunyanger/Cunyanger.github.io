### 修饰

#### 上下标

| 效果               | 代码               | 说明                       |
| ------------------ | ------------------ | -------------------------- |
| $a^2$              | `a^2`              | 上标（用 `^`）             |
| $a_2$              | `a_2`              | 下标（用 `_`）             |
| $a^2_3$ 或 $a_3^2$ | `a^2_3` 或 `a_3^2` | 同时上下标，顺序无关       |
| $a^{10}$           | `a^{10}`           | 多位数或多字符必须加花括号 |
| $a_{ij}$           | `a_{ij}`           | 多字符下标                 |
| $a^{x+y}_{i,j}$    | `a^{x+y}_{i,j}`    | 复杂表达式                 |

#### 矢量

- 单个字母矢量： `\vec` 即可（vector）
  `$\vec{i}$` or `$\vec i$` ： $\vec{i}$
- 多个字母就不要用空格了
  `$\vec{ij}$` : $\vec{ij}$\
  `$\vec ij$` ： $\vec ij$
- 比较复杂的左右箭头
  `$\overrightarrow x$` : $\overrightarrow x$
  `$\overrightarrow {x}$` : $\overrightarrow {x}$\
  `$\overleftarrow x$` : $\overleftarrow x$
  `$\overleftarrow {x}$` : $\overleftarrow {x}$
